About Me

My name is Yawo Sadji and I have a strong passion for software development. In November 2023, I took a break from my hybrid Field Tech/Tech Support role to focus on becoming a fullstack SWE. I learned HTML, CSS, JavaScript, and Core Java, and was selected for the 2024-2025 Verizon Thrive Apprenticeship Program by Multiverse. Over the past 12 months, I've built projects across different stacks, collaborated with other apprentices, and worked on company projects with experienced developers and mentors. This journey has helped me grow tremendously as a Fullstack SWE.

Overview of my Apprenticeship

  • Host Company: Verizon
  • Duration: 12 months
  • Portfolio Website (Coming Soon)

During my apprenticeship at Verizon, I worked on Java/Springboot codebases for the SOX compliance team, focusing on API integration, data troubleshooting, and improving process efficiency. I also learned RPA development using UiPath, transforming business requirements into PDDs, building automations with the ReFramework, and writing tests. My biggest impact was reducing manual effort and risk of human error, saving thousands of hours for my team.

Major Projects

Project 1: Titan Store (Inventory Fullstack Application)

View Repo

Titan Store is a shop application where users can browse items, view details, and manage inventory. Features include search, CRUD operations, and a responsive UI.

  • Frontend: React, Material UI
  • Backend: Node.js, Express.js
  • Database: SQLite (Sequelize ORM)
  • Testing: Jest, Supertest
  • Product catalog & details
  • Search functionality
  • Add, edit, delete items
  • RESTful API & automated tests
Reflections: Deepened React skills, learned fullstack project structure, and improved teamwork and communication.

Project 2: TEAMCRM (CRM Website for Small Teams)

View Repo

TeamCRM is a Django web app for small teams to manage contacts, with authentication, CRUD, search, and CSV export.

  • Backend: Django (Python)
  • Frontend: Bootstrap
  • Database: PostgreSQL
  • User authentication
  • Contact CRUD & search
  • Export contacts to CSV
  • Responsive UI
Reflections: Learned Django fundamentals, MVT pattern, database integration, and secure user authentication.

Project 3: DjangoStore (E-commerce Store API)

View Repo

DjangoStore is a RESTful API for e-commerce, supporting user authentication, product management, and order handling.

  • Backend: Django, Django REST Framework
  • Authentication: JWT
  • Database: SQLite
  • User & admin authentication
  • Product CRUD
  • Order management
  • API docs (Swagger/OpenAPI)
Reflections: Gained experience with REST APIs, JWT, API docs, and teamwork in shared codebases.

Project 4: SOX-Chatbot (AI Chatbot for SOX Compliance)

View Repo

SOX-Chatbot is a React app using Gemini AI and Firebase to help users learn about SOX compliance, CPRA, and Segregation of Duties.

  • Frontend: React, Vite, React Bootstrap
  • AI: Google Gemini
  • Database/Auth: Firebase
  • AI-powered chatbot
  • Google authentication
  • Chat history & real-time updates
  • Unit tested & responsive UI
Reflections: Improved React, AI integration, Firebase, and automated testing skills.

Project 5: RestaurantChatbot (AI Restaurant Chatbot)

View Repo

RestaurantChatbot lets customers place orders via an AI chatbot, with Google authentication, admin dashboard, and real-time updates.

  • Frontend: React, Vite, React Bootstrap
  • AI: Google Gemini
